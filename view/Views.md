# Viewers do framework

- lista
  - `pkit/code` : pagina que mostra um svg animado com base no codigo
    - argumentos
      - lang
      - title
      - code
      - color
      - message
  - `pkit/redirect` : pagina que mostra uma tela de carregamento animado e ainda redireciona para o site pedido
    - argumentos
      - lang
      - title
      - site
      - color
